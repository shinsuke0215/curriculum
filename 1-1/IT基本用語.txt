IT
コンピュータとネットワークを利用した情報技術

OS
アプリやデバイスを動作させるための基本的な機能を制御しているソフトウェア

Windows
マイクロソフトが開発したOS

macOS
Macが開発したOS

UNIX
現存する中で最も古いOSの一つ。
安全性が高く、余計な機能がなく軽く安いのが特徴。

Linux
無料で改変可能なOS
安定的に動いて、費用がかからないのが特徴。

HTML
Webページを作る上で文書構造を形作る為のマークアップ言語。

CSS
Webページを作る上で文書構造にデザインを施し、見栄えを整える役割を担うマークアップ言語。

MySQL
データベース管理システム。

PHP
動的にWebページを生成することができるサーバーサイドのスクリプト言語。

JavaScript
Webページを作る上で動的なページを作成する事のできるプログラミング言語。

jQuery
JavaScriptでできることを、より簡単に実現できるように設計されたJavaScriptライブラリ。

Apache
オープンソースで使えるWebサーバーソフトウェア。

Bootstrap
CSSのフレームワーク。よく使われるスタイルがあらかじめ定義してある。
ルールに沿って利用するだけで整ったデザインを作成可能。

SE
必要なプロジェクトに基づき、最適な仕様のシステム開発を行う職種。

PG
Webシステムや業務システムなどに必要な機能を、プログラミング言語を使って実装する職種。

Webデザイナー
クライアントから依頼されたWebサイトのデザインを作る職業。

CUI
キーボードに入力されるコマンドでOSの操作をする。

GUI
コンピューターの画面上に表示されるウィンドウやアイコン、ボタンなどを使い、マウスなどで操作する。

ファイル
データのやりとりを行うためのデータのまとまり。

ディレクトリ
フォルダのこと。

ルート・ディレクトリ
ファイル階層の最初のディレクトリのこと。

カレント・ディレクトリ
現在自分がいるディレクトリのこと。

パス
ファイルや、フォルダがおいてある場所を示す。

絶対パス
ルートディレクトリから目的地までの経路を表す。

相対パス
現在作業してるフォルダから目的のフォルダまでの経路を表す。

HTTP
ホームページのファイルなどを受け渡しするときに使う決まりごと。

FTP
ファイルを送受信する通信の決まりごと。

HTTPS
インターネットで暗号化通信するときの決まりごと。通信をするときに暗号化される。

SSL
インターネット上で情報を暗号化して送受信するための仕組み。

SSH
ネットワークを経由して他のコンピュータに接続して、遠隔操作する仕組み。

ポート
通信において送受信する際に、データが通る接続口。

ポート番号
通信において送受信する際に、データが通る接続口に割り振られた識別用番号。

HTTP、FTP、SSHの各ポート番号
HTTP⇨80、FTP⇨20、21、SSH⇨22

ファイアウォール
ネットワークから侵入してくる不正なアクセスから守るためのもの。

TCP
送ったデータが相手に届いたか確認しながら通信するやり方。通信プロトコルの一つ。

IPアドレス
インターネットをする上でコンピュータに割り当てられる住所のようなもの。

グローバルIPアドレス
全世界で通用するIPアドレスのこと。

プライベートIPアドレス
身内内、仲間内でしか通用しないIPアドレスのこと。自分たちで決めれるIPアドレス。

ドメイン
ドメイン名のこと。IPアドレスにつけた名前。「http://」の後ろの部分のこと。

DNS
IPアドレスとドメイン名の紐付けをしてくれる役割。通訳みたいな役割。

VPN
普通のインターネット回線などを使いながらでも専用回線のようにしてくれるサービス。

ブリッジ
MACアドレスを見てデータのの行き先を振り分けてくれる教えてくれる機能。

プロキシサーバー
ホームページなどを見る際に、Webブラウザとサーバでやり取りする時の中間役。

SMTP
メール送信時に使う決まりごと。通信プロトコルの一つ。

POP
自分が使ってるメールサーバに届いたメールを自分のPCにコピーしてきて見るときに使う決まりごと。通信プロトコルの一つ。

IMAP
メールサーバに直接お邪魔して、入ってるメールを直接見るやり方。

セッション
論理的な意味で開始から終了までの一連の流れ。

クッキー
WebサーバがWebブラウザに渡すメモがきのようなもの。状態などを維持、管理するための仕組み。

キャッシュ
ホームページの受け渡しの際にPCがファイルのコピーをしておいて予め準備しておく仕組み

チューニング
欠点だったりなどを状態が良くなるように調整すること。

スロークエリ
データベースで時間のかかるSQL（データベースとのやりとり）のこと

インデックス
検索速度を速くする楽にするための情報。

フロー
作業を手順に追った流れのこと。

単体テスト
部分部分の動くかどうかの確認のテスト。

結合テスト
作ったシステムがちゃんと動いて問題ないか確認するテスト。

チケット管理ツール
発生した各種タスクを制作者に仕事を依頼するためのタスク管理ツール。

タスク
やるべき仕事。

Git
ファイルの変更履歴を管理してくれるシステム。開発者自信のPCにローカルリポジトリを持ちこれに修正追加したものをコミットする。

SourceTree
GitをGUIで扱うためのアプリケーションの一つ。

SVN
ファイルの変更履歴を管理してくれるシステム。サーバーのリモートリポジトリにアクセスして自分のPCに指定したバージョンをダウンロード。修正や追加したコードはリモートリポジトリにコミットする。

コンフリクト
複数のプログラムが同じデータやファイルを同時に使おうとしている状態。

デグレ
変更したことにより、今までできていたことができなくなった時の状態。

マージツール
視覚的にソースコードやファイルなどを編集、開発する段階で比較、新たにコードを追加したり、修正を反映させたりできる。
